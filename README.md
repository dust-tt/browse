# Browse

A TypeScript-based web automation tool consisting of a CLI (`wb`) and a server daemon (`wbd`).

## Installation

```bash
npm install
```

## Building

```bash
# Build CLI
npm run build:cli

# Build daemon
npm run build:daemon

# Build both
npm run build
```

## Development

```bash
# Start daemon in dev mode
npm run dev:wbd

# Run CLI in dev mode
npm run dev:wb
```
